## 📱 Ekran Görüntüsü Alma Rehberi / Screenshot Guide

### Android
1. Emülatör/Cihazda uygulamayı çalıştırın
2. İlgili sayfaya gidin
3. `Cmd + S` (Mac) veya `Ctrl + S` (Windows) - Emülatörde
4. Veya cihazda: `Volume Down + Power` tuşlarına aynı anda basın

### iOS
1. Simulator'da uygulamayı çalıştırın
2. İlgili sayfaya gidin
3. `Cmd + S` - Simulator'da screenshot alır
4. Veya: iOS Device > Capture Screen

### Önerilen Boyutlar / Recommended Sizes
- Genişlik / Width: 1080px - 1440px
- Yükseklik / Height: Cihaza göre / Device dependent
- Format: PNG (en iyi kalite için / for best quality)

## 🎨 Düzenleme İpuçları / Editing Tips

- Durum çubuğunu temizleyin (Demo Mode kullanın)
- Saat göstergesini 9:41'e ayarlayın (Apple standardı)
- Batarya göstergesini %100 yapın
- Bildirim simgelerini temizleyin
- Tutarlı bir tema kullanın (light/dark)
