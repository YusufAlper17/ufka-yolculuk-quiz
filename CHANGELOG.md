## [1.0.0] - 2025-11-XX

### Eklenen / Added
- 🎯 4 farklı kategori (İlkokul, Ortaokul, Lise, Yetişkin)
- 📚 AI destekli zengin soru havuzu
- 🏆 İlerleme takip sistemi
- 🎚️ 3 zorluk seviyesi (Kolay, Orta, Zor)
- ⏱️ Zamanlı quiz modu
- 🎨 Modern ve şık kullanıcı arayüzü
- 🔊 Ses efektleri
- 📊 Detaylı sınav raporları
- 🌓 Koyu/Açık tema desteği
- 📱 Android ve iOS desteği
- 💾 Offline çalışma özelliği
- 🎯 Her kategori için özel icon'lar
- ⭐ Smooth animasyonlar ve geçişler
- 📈 Skor ve ilerleme takibi
- 🔒 Seviye kilitleme/açma sistemi

### Teknik / Technical
- Flutter 3.2.3+ ile geliştirildi
- Provider state management
- SharedPreferences için yerel depolama
- Google Mobile Ads entegrasyonu
- AudioPlayers ses sistemi
- Animate Do animasyon kütüphanesi
- Shimmer loading efektleri
- Google Fonts (Poppins, Inter)

## Versiyon Türleri / Version Types

- **Deprecated** - Yakında kaldırılacak özellikler / Soon-to-be removed features
- **Removed** - Kaldırılan özellikler / Removed features
- **Fixed** - Hata düzeltmeleri / Bug fixes
- **Security** - Güvenlik güncellemeleri / Security updates
